LoRA Image Generation

Trained a diffusion model using LoRA to generate custom images of a specific person from just a few photos.

Description

This project fine-tunes a Stable Diffusion model using LoRA (Low-Rank Adaptation) to generate realistic and stylized images of a specific person. Using as few as 10-15 reference images, the model learns to recreate and synthesize new images of the subject in various poses, settings, and styles.

Key Features

  • Fine-tuned Stable Diffusion using PEFT (LoRA)
  • Learns a subject's appearance with few-shot training (10–15 images)
  • Generates high-quality custom images in multiple prompts/styles
  • Training handled via DreamBooth pipeline with LoRA for efficiency

Workflow

  • Collect ~15 photos of a subject (same person, different angles)
  • Preprocess images and caption using BLIP or manual annotations
  • Set up DreamBooth-style training with LoRA on Stable Diffusion
  • Fine-tune using a unique token like "sks person"
  • Generate images using prompts like "a photo of sks person in a forest"

Technical Stack

  • Language:Python
  • Libraries:Diffusers, PEFT, Hugging Face Accelerate, Transformers
  • Model:Stable Diffusion v1.5 with LoRA

Use Cases

  • Custom portrait generation from limited data
  • Research into efficient model personalization
  • Game or avatar character creation
  • Personalized content for social media or branding

Medicinal Plant Classification

Developed a machine learning model to classify medicinal plants based on leaf images using traditional feature extraction techniques.

Description

This project focuses on identifying various medicinal plants through images by employing machine learning techniques. The process involves segmentation, grayscale conversion, and advanced feature extraction methods such as GLCM, Gabor, and LBP. Multiple models are trained and evaluated to identify the most suitable one for prediction.

Key Features

  • Segmentation using HSV color space to isolate plant regions
  • Grayscale conversion for simplified processing
  • Feature extraction using LBP, GLCM, Gabor filters, and color moments
  • Dimensionality reduction with PCA and normalization with StandardScaler
  • Model training with various algorithms, achieving 99% accuracy with SVM

Workflow

  • Collect and label images of 30 different medicinal plant species
  • Segment images to isolate plant regions using HSV color space
  • Convert segmented images to grayscale
  • Apply Sobel filter to highlight edges
  • Extract features using LBP, GLCM, Gabor filters, and color moments
  • Reduce dimensionality with PCA and normalize features with StandardScaler
  • Train and evaluate multiple models, selecting the best-performing one
  • Use the trained model for real-time prediction on new data

Technical Stack

  • Language: Python
  • Libraries: OpenCV, scikit-learn, NumPy, Pandas
  • Techniques: Image segmentation, feature extraction, dimensionality reduction, machine learning classification

Use Cases

  • Assisting botanists and researchers in identifying medicinal plants
  • Developing mobile applications for plant identification
  • Supporting traditional medicine practices by identifying plant species
  • Educational tools for teaching plant taxonomy and machine learning
